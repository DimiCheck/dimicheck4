DimiCheck 채팅 개인정보 처리방침

시행일: 2025.11.26 버전: v1.0 문의: hjun1052@dimigo.hs.kr

채팅·투표·이모티콘 등 학급 커뮤니티 기능(“채팅 서비스”)에서 처리되는 개인정보 사항을 안내합니다. 기본 개인정보처리방침과 함께 적용됩니다.

1. 수집 항목

  • 식별: 학년, 반, 번호, 사용자 유형(학생/교사/보드), OAuth 식별자
  • 채팅 기록: 메시지 내용, 이미지 URL, 답장 관계, 삭제 여부, 반응, GIF/이모티콘 사용 내역
  • 투표: 질문/옵션/응답, 종료 시각, 내 투표 선택
  • 프로필/아바타/닉네임 설정, 채널 정보(소유자, 참여 반)
  • 기기·기록: 접속 로그(시간/브라우저/OS), 알림 토큰, 쿠키·로컬스토리지 저장 데이터

2. 이용 목적

  • 학급 커뮤니케이션 제공(채팅, 투표, 반응, 이모티콘)
  • 출결·현황판 연동, 사용자 프로필 표시, 부정 이용 방지
  • 알림 제공, 보안 로그 기록, 장애 대응 및 품질 개선

3. 보유 기간

  • 채팅·투표 기록: 학사연도 내 보관 후 정기 삭제
  • 알림 토큰/세션: 서비스 제공에 필요한 기간(만료 시 자동 삭제)
  • 법령상 보존 의무가 있는 경우 해당 기간 동안 보관 후 파기

4. 제3자 제공·위탁

원칙적으로 제3자에게 제공하지 않으며, 이미지 업로드 등 외부 스토리지 사용 시 필요한 범위 내에서만 처리합니다. 제공·위탁 시 별도 고지·동의를 받습니다.

5. 이용자 권리

  • 채팅 메시지·이모티콘 직접 삭제, 닉네임/아바타 수정
  • 이메일을 통한 열람·정정·삭제·동의철회 요청(법정대리인 포함)

6. 안전조치

  • 세션 보호, 비밀번호 미수집, HTTPS 통신
  • 권한 분리(학생/교사/보드), 채널 소유자 기반 관리
  • 주기적 로그 점검 및 학사연도 단위 데이터 정리
